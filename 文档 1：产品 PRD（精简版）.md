# 文档 1：产品 PRD（精简版）

## 1. 产品定义

训练间歇记录器是一个极简力量训练记录工具。

当前实现主要解决三个问题：

1. 今天练什么
2. 每个动作还要休息多久
3. 刚刚这一组做了什么

------

## 2. 目标用户

适用于：

- 做力量训练的人
- 有按组训练习惯的人
- 需要记录组数、重量、次数、时长或距离的人
- 觉得传统健身记录 App 太重的人

不适用于：

- 有氧训练用户
- 只想打卡的人
- 需要复杂训练计划的人
- 需要社区、教程、课程的人

------

## 3. 核心原则

### 3.1 训练中只有一个核心操作

**点击一次 = 完成当前动作的一组**

点击后系统自动：

- 记录该组完成
- 记录该组完成时间
- 启动该动作组间歇倒计时
- 更新动作状态

最近一组的补录入口保留，根据动作记录类型展示重量、次数、时长或距离输入，由用户手动打开，不在完成本组后自动弹出。

### 3.2 训练不是线性流程

系统不强制用户按动作顺序训练。
用户可以在不同动作之间自由切换。

### 3.3 训练安排页是今日训练总控页

训练安排页用于查看今日训练的所有动作状态、倒计时，并切换进入不同动作。

### 3.4 倒计时只负责提醒

倒计时只用于提示休息。
不会自动进入下一组，也不会自动切换到下一个动作。
用户离开动作页后，倒计时继续运行。

### 3.5 计划只是默认动作集合来源

- 训练计划：长期维护的默认动作集合
- 今日训练：按天存在的一份实际训练安排，由当天首次进入时自动创建

从计划导入动作时，只把计划中的默认动作追加到今日训练，不替换今日训练已有动作。
计划导入弹层允许只勾选其中部分动作；默认全选，保持“加入这个计划”这一既有语义。
训练中手动新增动作，只影响今日训练，不写入计划。
计划编辑必须在独立页面完成，且不会自动同步到已经生成的今日训练或历史训练。

------

## 4. MVP 功能范围

### 当前已实现

- 在训练安排页查看今日训练
- 从计划追加默认动作到今日训练，并允许只选择其中部分动作
- 在计划中为动作配置可选默认重量/次数
- 进入任一动作页
- 点击完成本组
- 记录每组完成时间
- 每个动作独立倒计时
- 按动作记录类型补录最近一组重量、次数、时长或距离
- 训练中自由切换动作
- 为今日训练手动新增动作
- 删除今日训练中未开始动作
- 独立计划编辑页
- 训练总结（当前展示日期、状态、动作完成情况与逐组记录）
- 本地数据持久化

### 当前明确不做 / 尚未实现

- 语音输入
- 自动识别重量或次数
- 自动推进下一组
- 自动切换动作
- 固定线性训练流程
- 社区 / 教程 / 课程
- 成就 / 排行榜
- 复杂统计分析
- 训练中直接编辑长期计划
- 独立历史列表
- 训练总时长统计

------

## 5. 核心业务规则

### 5.1 首组记录规则

进入动作页后，不自动开始当前组计时。
第 1 组只有在第一次点击“完成本组”时才生成组记录，并写入该组 `completedAt`。

### 5.2 动作开始规则

进入动作页不算动作已开始。
只有第一次点击“完成本组”后，该动作才从“未开始”变为“进行中”。

### 5.3 退出动作页规则

若进入动作页后未点击“完成本组”就离开：

- 不生成组记录
- 不更新动作状态
- 不启动倒计时

### 5.4 今日训练追加动作

允许从计划导入默认动作，也允许手动新增动作。
新增或导入都只追加到今日训练，不替换已有动作，也不写入长期计划。

### 5.5 训练中删除动作

仅允许删除今日训练中“未开始”的动作。
进行中或已完成动作不可删除。

### 5.6 计划导入规则

- 计划不是今日训练的锁定来源
- 从计划导入动作是追加，不替换今日训练已有动作
- 导入前允许只选择计划中的部分动作
- 训练开始后仍可继续从计划追加动作
- 若今日训练已经完成，再追加动作前需要用户确认继续训练

### 5.7 计划默认值规则

- 计划动作可选配置默认 `weightKg` 与 `reps`
- 当今日训练中的动作使用重量次数记录时，完成某组后，新组记录会先复制计划默认值作为初始内容
- 用户随后在补录中修改的，是本次训练的真实记录，不会反向修改计划默认值

### 5.8 动作记录类型规则

- `weightReps` 记录重量与次数
- `reps` 记录次数
- `duration` 记录时长
- `distance` 记录距离
- `weightDistance` 记录重量与距离
- 记录页、最近一组展示和总结指标按动作记录类型显示对应字段

### 5.9 训练完成规则

当今日训练中的所有动作都达到目标组数时，今日训练完成。
