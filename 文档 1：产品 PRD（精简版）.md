# 文档 1：产品 PRD（精简版）

## 1. 产品定义

训练间歇记录器是一个极简力量训练记录工具。

MVP 只解决三个问题：

1. 本次练什么
2. 每个动作还要休息多久
3. 刚刚这一组做了什么

------

## 2. 目标用户

适用于：

- 做力量训练的人
- 有按组训练习惯的人
- 需要记录组数、重量、次数的人
- 觉得传统健身记录 App 太重的人

不适用于：

- 有氧训练用户
- 只想打卡的人
- 需要复杂训练计划的人
- 需要社区、教程、课程的人

------

## 3. 核心原则

### 3.1 训练中只有一个核心操作

**点击一次 = 完成当前动作的一组**

点击后系统自动：

- 记录该组完成
- 计算该组用时
- 启动该动作组间歇倒计时
- 更新动作状态

### 3.2 训练不是线性流程

系统不强制用户按动作顺序训练。
用户可以在不同动作之间自由切换。

### 3.3 训练安排页是本次训练总控页

训练安排页用于查看本次训练的所有动作状态、倒计时，并切换进入不同动作。

### 3.4 倒计时只负责提醒

倒计时只用于提示休息。
不会自动进入下一组，也不会自动切换到下一个动作。
用户离开动作页后，倒计时继续运行。

### 3.5 模板与本次训练分离

- 训练模板：长期默认配置
- 本次训练：基于模板生成的一次具体训练

训练中临时新增动作，只影响本次训练，不写入模板。
模板编辑必须在独立页面完成。

------

## 4. MVP 功能范围

### 必须有

- 在训练安排页选择模板
- 查看本次训练动作列表
- 进入任一动作页
- 点击完成本组
- 自动计算组用时
- 每个动作独立倒计时
- 补录最近一组重量/次数
- 训练中自由切换动作
- 为本次训练新增动作
- 删除本次训练中未开始动作
- 独立模板编辑页
- 训练总结
- 历史记录保存

### 明确不做

- 语音输入
- 自动识别重量或次数
- 自动推进下一组
- 自动切换动作
- 固定线性训练流程
- 社区 / 教程 / 课程
- 成就 / 排行榜
- 复杂统计分析
- 撤销“完成本组”
- 训练中直接编辑长期模板

------

## 5. 核心业务规则

### 5.1 首组计时规则

进入动作页后，开始当前待完成组计时。
第 1 组用时 = 进入动作页时刻 到 第一次点击“完成本组”时刻。

### 5.2 动作开始规则

进入动作页不算动作已开始。
只有第一次点击“完成本组”后，该动作才从“未开始”变为“进行中”。

### 5.3 退出动作页规则

若进入动作页后未点击“完成本组”就离开：

- 不生成组记录
- 不更新动作状态
- 不启动倒计时
- 当前这次计时作废

### 5.4 训练中新增动作

允许新增，但只加入本次训练，不写入长期模板。

### 5.5 训练中删除动作

仅允许删除本次训练中“未开始”的动作。
进行中或已完成动作不可删除。

### 5.6 模板切换规则

- 本次训练未开始时：允许切换模板
- 本次训练已开始时：不允许切换模板

### 5.7 训练完成规则

当本次训练中的所有动作都达到目标组数时，本次训练完成。
