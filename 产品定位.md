一个极简力量训练记录器：先快速开始今天的训练，再把常用训练保存为模板。

你现在这个产品，**不要再描述成 timer**。

因为用户一听 timer，会自动理解成：

> 设置训练时间、休息时间、循环次数，然后自动播放倒计时。

但你的首页已经很清楚了：它不是一个“播放式计时器”，而是一个**力量训练现场控制台**。

你的产品核心其实是：

> 今天练哪些动作、每个动作做到第几组、哪个动作正在休息、下一组什么时候可以开始。

所以一句话应该是：

> **A simple workout set tracker with automatic rest timers.**

中文就是：

> **一个带自动组间计时的极简力量训练记录器。**

这是目前最准确、最容易让人秒懂的描述。

------

## 我建议的产品定位

你的产品不是：

> Training Timer
> Interval Timer
> Workout Timer
> Gym Timer

而是：

> Set Tracker
> Rest Timer
> Workout Log
> Strength Training Tracker

更准确地说：

> **Set-based strength training tracker**

也就是“按组驱动的力量训练记录器”。

## 你的产品真正该强调的三个点

从你首页来看，最应该强调的是：

### 1. One tap per set

用户不需要填一堆东西。

> 完成一组，点一下。

### 2. Automatic rest timer

点完后自动开始对应动作的休息倒计时。

> 不需要手动开计时器。

### 3. Multiple exercises, non-linear training

用户可以在 Pull-up、Seated row、Face pull 之间自由切换。

> 不强迫线性训练流程。

这一点是你和普通 interval timer 最大的区别。

- 每个动作完成情况
- Ready / Resting / Not started
- 每个动作独立状态

这句话很好。
