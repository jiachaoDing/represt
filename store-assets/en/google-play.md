# Google Play Assets

## App Name
RepRest: Set & Rest Tracker

## Short Description
Track each set and start your rest timer with one tap.

## Full Description
RepRest is a simple set tracker for strength training. Start today's workout quickly, tap once when you finish a set, and let the app start the rest timer automatically.

It is built for real training sessions where you may move between multiple exercises instead of following a fixed interval timer. See what is ready, what is resting, and what still needs work.

Core features:
- One-tap set tracking
- Automatic rest timers for each exercise
- Today's workout progress at a glance
- Flexible exercise order during training
- Workout templates for repeated routines
- Set, weight, and rep history
- Local rest reminders and haptic feedback

## Data Safety Positioning
Training records, plans, and settings are stored primarily on the device. The app does not require account sign-in and does not proactively upload training data.

## Notification / Exact Reminder Permission Explanation
Notifications remind you to continue training when a rest countdown ends. Exact reminder permission improves rest timer punctuality.
