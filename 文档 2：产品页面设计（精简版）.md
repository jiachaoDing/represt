# 文档 2：产品页面设计（精简版）

## 页面 1：训练安排页

### 页面作用

今日训练总控页。

### 显示内容

- 今日训练归属日
- 今日训练状态
- 模板动作导入入口
- 动作列表

每个动作显示：

- 动作名
- 目标组数
- 组间歇时长
- 已完成组数
- 动作状态
- 剩余倒计时 / 可继续下一组

### 支持操作

- 从模板追加默认动作到今日训练
- 在模板弹层中勾选部分动作后加入今日训练
- 点击动作进入动作页
- 手动新增今日训练动作
- 删除今日训练中未开始动作
- 进入模板编辑页
- 在训练完成后从菜单进入训练总结页

### 页面跳转

- 点击动作 → 动作页
- 点击模板标签 → 打开模板预览，可勾选部分动作后追加到今日训练
- 点击菜单中的模板编辑入口 → 模板编辑页
- 点击菜单中的训练总结入口 → 训练总结页

### 当前补充规则

- 今日训练已完成时，继续追加模板动作前需要确认
- 今日训练已完成时，继续手动新增动作前需要确认
- 当没有模板时，页面仍允许直接手动新增动作
- 模板弹层默认全选全部动作；若一个动作都没选，则禁止提交并提示至少选择 1 个动作

## 页面 2：动作页

### 页面作用

单个动作执行页。

### 显示内容

- 动作名
- 当前进度（已完成组数 / 总组数）
- 动作状态
- 完成本组按钮
- 当前动作剩余倒计时
- 最近一组记录
  - 组序号
  - 完成时间
  - 重量
  - 次数
- 重量输入框
- 次数输入框
- 返回训练安排页

### 支持操作

- 点击“完成本组”
- 手动打开最近一组补录弹层
- 补录最近一组重量
- 补录最近一组次数
- 返回训练安排页

### 页面规则

- 进入页面后，不自动开始当前组计时
- 点击后页面原地更新，不自动跳转
- 已完成动作不可继续点击“完成本组”
- 每个动作页只补录本动作最近一组
- 完成一组后不自动打开最近一组补录弹层
- 若动作来自模板且模板配置了默认重量/次数，则最近一组补录表单默认带入这些初始值

### 页面跳转

- 返回训练安排页 → 训练安排页

------

## 页面 3：模板编辑页

### 页面作用

长期训练模板维护页。

### 显示内容

- 模板列表
- 当前模板名称
- 模板动作列表

每个动作显示：

- 动作名
- 默认组数
- 默认组间歇时长
- 可选默认重量
- 可选默认次数

### 支持操作

- 新增模板
- 编辑模板名称
- 删除模板
- 新增模板动作
- 编辑模板动作
- 删除模板动作
- 返回训练安排页

### 页面规则

- 这里只维护长期模板
- 不展示训练实时状态
- 不展示倒计时
- 已开始的训练不会因模板编辑自动同步变化

### 页面跳转

- 返回训练安排页 → 训练安排页

### 当前补充规则

- 模板页通过标签切换当前模板
- 模板动作点击进入编辑弹层，左滑可删除，长按打开上下文菜单

------

## 页面 4：训练总结页

### 页面作用

展示本次训练结果。

### 显示内容

- 训练日期
- 当前训练状态
- 完成动作数
- 动作汇总列表

每个动作显示：

- 动作名
- 完成组数
- 每组记录
  - 组序号
  - 完成时间
  - 重量
  - 次数

### 支持操作

- 查看总结
- 返回训练安排页

### 当前未展示

- 训练总时长
- 历史训练列表入口

### 页面跳转

- 返回训练安排页 → 训练安排页
